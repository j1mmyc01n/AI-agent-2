Current Issue: "Application error: a server-side exception has occurred"
Root Cause:     Missing environment variables in Netlify

┌──────────────────────────────────────────────────────────────────────────┐
│ ✅ STEP 1: SET THESE 3 VARIABLES IN NETLIFY                              │
└──────────────────────────────────────────────────────────────────────────┘

Go to: https://app.netlify.com → Your Site → Site Settings → Environment Variables

┌─────────────────────────┬────────────────────────────────────────────────┐
│ Variable Name           │ How to Get the Value                           │
├─────────────────────────┼────────────────────────────────────────────────┤
│ DATABASE_URL            │ 1. Go to https://neon.tech                     │
│                         │ 2. Create free account + project               │
│                         │ 3. Copy connection string                      │
│                         │    (starts with postgresql://)                 │
├─────────────────────────┼────────────────────────────────────────────────┤
│ NEXTAUTH_SECRET         │ Run in terminal:                               │
│                         │ openssl rand -base64 32                        │
│                         │ Copy the output (32+ characters)               │
├─────────────────────────┼────────────────────────────────────────────────┤
│ NEXTAUTH_URL            │ Your Netlify site URL                          │
│                         │ Example: https://dobetteragent2.netlify.app    │
│                         │ (Find in Netlify dashboard)                    │
└─────────────────────────┴────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────┐
│ ✅ STEP 2: INITIALIZE DATABASE                                           │
└──────────────────────────────────────────────────────────────────────────┘

Run this on your LOCAL MACHINE (replace with your actual DATABASE_URL):

DATABASE_URL="postgresql://user:pass@host.neon.tech/db" npx prisma db push

┌──────────────────────────────────────────────────────────────────────────┐
│ ✅ STEP 3: REDEPLOY                                                      │
└──────────────────────────────────────────────────────────────────────────┘

Go to Netlify dashboard:
1. Click "Deploys" tab
2. Click "Trigger deploy" → "Clear cache and deploy site"
3. Wait ~2 minutes for build to complete
4. Visit your site - it should work! 🎉

┌──────────────────────────────────────────────────────────────────────────┐
│ 🆘 STILL NOT WORKING?                                                    │
└──────────────────────────────────────────────────────────────────────────┘

1. Check Netlify Function Logs:
   Netlify Dashboard → Functions → Click any function → See error message

2. Look for these specific errors:
   ❌ DATABASE_URL environment variable is not set!
   ❌ NEXTAUTH_SECRET environment variable is not set!
   ❌ NEXTAUTH_URL environment variable is not set!

3. Double-check your environment variables:
   - DATABASE_URL starts with postgresql:// (not postgres://)
   - NEXTAUTH_SECRET is at least 32 characters
   - NEXTAUTH_URL matches your exact Netlify URL (no trailing slash)

4. Run validation script locally:
   npm run check-env

5. See full guide:
   https://github.com/j1mmyc01n/AI-agent-2/blob/claude/fix-netlify-build-issues/SETUP_ENVIRONMENT.md

┌──────────────────────────────────────────────────────────────────────────┐
│ 📋 OPTIONAL VARIABLES (for extra features)                              │
└──────────────────────────────────────────────────────────────────────────┘

Add these in Netlify for additional functionality:

• OPENAI_API_KEY        - AI chat features (get from platform.openai.com)
• GITHUB_ID             - GitHub OAuth login
• GITHUB_SECRET         - GitHub OAuth login
• TAVILY_API_KEY        - Web search in AI (get from tavily.com)

Without these, the app still works but some features will be disabled.
